# MindType - MBTI Personality Analysis

🧠 Discover your 32-type MBTI personality through scientific tests.

## Features

- **Dual Test Modes**: Quick (25Q, ~5min) & Full (105Q, ~20min)
- **32-Type Analysis**: Classic 4 dimensions + A/T fifth dimension
- **8-Module Report**: Overview, Strengths, Relationships, Career, Stress, Cognitive Functions, Famous Types, Comparison
- **64 Avatars**: Gender-specific illustrations for all 32 types
- **Test History**: Save and review past results with dates
- **Social Sharing**: Share results to Facebook, Instagram, and X
- **4 Languages**: 简体中文 / 繁體中文 / English / 日本語
- **Randomized Questions**: Fisher-Yates shuffle for unique experience every time

## Deployment

This is a single-page static website. Just serve `index.html`.

### GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Source to **Deploy from a branch**
4. Select **main** branch, root folder
5. Save — your site will be live at `https://YOUR_USERNAME.github.io/REPO_NAME/`

## Tech Stack

- React 18 (CDN)
- Babel Standalone (CDN)
- Pure CSS (no framework)
- 64 embedded base64 avatar images
